## Installation

1. **`git clone https://github.com/luisedware/CowCat.git`**
1. **`cd CowCat`**
1. **`sudo chmod -R 777 storage/`**
1. **`sudo composer install`**
1. **`sudo npm install`**
1. **`sudo vi .env`**
1. **`php artisan migrate:refresh --seed`**
1. **`php artisan serve`**
1. **`gulp watch`**
